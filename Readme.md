# NixOS babysteps in CrossCompiling / image building

An ongoing effort porting NixOS to the RISC-V architecture through cross compilation

## Supported Boards

### VisionFive 2

#### Build
```bash
nix build .#visionfive2-sdcard
```


### RaspberryPi 4

#### Build
```bash
nix build .#rpi4-sdcard
```
